# AI-Powered Educational Chatbot

## Description
This project is a simple AI-based chatbot designed to answer admission and
scholarship-related queries for educational institutions.

## Features
- Answers admission queries
- Provides scholarship information
- Basic NLP-based intent detection
- Multilingual support (conceptual)

## Technologies Used
- Python
- Natural Language Processing (NLP)

## Outcome
Improves accessibility to educational information and reduces manual effort.
